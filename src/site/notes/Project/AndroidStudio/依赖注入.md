# 介绍
## 依赖 ：类的初始化
### 初始化方式 ：
1. 类自身初始化
2. 其他对象帮助
## 依赖注入 ：通过其他的对象帮助完成初始化
### 谷歌提供的两套框架
1. Dagger：提供依赖注入，复杂一点
2. Hilt：基于Dagger做的，场景化
### 依赖注入的方式
1. 构造函数注入：通过构造函数将依赖对象传递
2. 方法注入：通过类的方法将依赖对象传递
3. 属性注入：通过类的属性将依赖对象传递
4. [[Project/AndroidStudio/注解\|注解]]：JAVA和Kotlin语言中
